What is the URL of this website?

The URL of this website is "https://www.greenjoecoffeeschool.com/"

What is the name of the website?

The website name is "Green Joe Coffee School - Online Learning"

Who is the site's target audience?

The target audience are people who want to learn how to start a mobile coffee business.

How is the site organized?

This website uses Hierarchical Organization.

Which CRAP Design Principle does the website use? Provide at least one example.

This website uses the Proximity principle.

In the screenshot below, you see these like items together in one section of the website. Each of these items are Navigation Links to specific resources that the page is pushing the most.

What is the Audit Score according to the Accessibility Checker?

This website received an Audit score of 69, which puts them at risk of an accessibility lawsuit.

What is the site's effectiveness? Does it support users in completing actions accurately?

This website allows for actions to be completed accurately. Every resource is carefully listed out and described in detail. They even include a Step By Step Checklist.

What is the site's efficiency? Can users perform tasks quickly?

This website allows for tasks to be performed fairly quickly. This website uses Navigation links on the top of the page, that you can hover over to quickly navigate to most sections of the website with one click.

How is the engagement? Is it pleasant to use and appropriate for its industry/topic?

This website is very versatile, as it lists out information in many different ways, from checklists to youtube videos. This website also provides plenty of resources for free, which is preferred considering the topic is educational.

Make at least one recommendation to improve this website based on what you learned in this module.

One recommendation I would make would be to ensure more contrast between the foreground and the background colors. The current contrast may be difficult for people with colorblindness or low vision disabilities.
